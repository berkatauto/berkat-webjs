Berkat Auto - Web JS.

Integrated With JSDelivr.

Updated:

The script have been upgraded to ES6+, which means there will be only one script that can be used for running many script at once.

How to Use:

1. Copy the javascript link :

http://cdn.jsdelivr.net/gh/berkatauto/berkat-webjs/main.js

2. Write a script as a module inside your html.

3. Paste your copied codes into your script and check it if it works.

Example:

This is an example how do you write your code inside the html...

script type="module" src="http://cdn.jsdelivr.net/gh/berkatauto/berkat-webjs/main.js"

Make sure to put the right element in HTML correctly.
